Bonjour à tous,

voici notre espace sur le serveur qui nous permettra de voir l'avancement de notre développement sur le projet MADERA.

Cet espace Web associé à un projet GITHUB.
Git est une application de versionning de fichiers permettant de travailler à plusieurs simultanément sur un même projet, sans pertes de données
et avec un versionning automatique accompagné de logs de changements.

GitHUB permet de stocker ce projet en ligne directement, proposant ainsi l'édition/création de fichiers online.
GitHUB permet aussi de faire des demandes de changements sur le projet via un système de ticket, permettant ainsi de tracer les demandes éffectuées.


Pour contribuer à ce projet, 2 solutions s'offrent à vous :
  • Éditer les fichiers directement via GITHUB.
  • Télécharger github for windows, puis éditer vos fichiers avec un éditeur tel que notepad ++




Dans chacun des cas, vous devez vous inscrire sur le site GITHUB.
